How it works

Five-stage pipeline

From raw FTIR output to automatic classification — every step built and understood, not just called from a library.

01 — COLLECT

⚗️

FTIR Spectroscopy

Shimadzu spectrometer. 400–4000 cm⁻¹. 5 repeats per compound. 380 spectra total from 76 compounds.

02 — CLEAN

∿

Interpolation + Normalisation

Integer wavenumber resampling. Trapezium-rule area normalisation. Fingerprint region extracted (630–880 cm⁻¹).

03 — REDUCE

⬡

Principal Component Analysis

251 wavenumber features compressed to 20 principal components. Class separation confirmed visually in PC-space.

04 — TRAIN

◎

k-NN Classifier

67:33 train/test split by compound (not spectrum). k=3 nearest neighbours. No data leakage.

05 — CLASSIFY

✦

Unknown Identification

5 personally collected unknown isomers classified against the full training library. 5/5 correct.

Implementation

Built from first principles

The trapezium rule, interpolation logic, and data-loading pipeline were all written from scratch before sklearn was introduced for PCA and classification. The goal was understanding, not just accuracy.

def load_spectra(n, spectra_dir="data/raw_ir_spectra"):
    spectra = []
    for entry in os.scandir(spectra_dir):
        df = pd.read_csv(entry.path, skiprows=4, sep=r"\s+",
                         names=[entry.name], index_col=0)
        df = normalisation(df)    # area → 1 via trapezium rule
        df = interpolation(df)    # decimal → integer wavenumbers
        df = narrow_range(df)     # keep 630–880 cm⁻¹ only

        name = re.sub(r'_\d+$', '', entry.name.removesuffix('.txt'))
        df.columns = [name]
        spectra.append(df)

    combined = pd.concat(spectra, axis=1)
    if n > 0:
        return perform_pca(n, combined)
    return combined
