How it works

Five-stage pipeline

From raw FTIR output to automatic classification — every step built and understood, not just called from a library.

01 — COLLECT

FTIR Spectroscopy

Shimadzu spectrometer. 400–4000 cm⁻¹. 5 repeats per compound. 380 spectra total.

02 — CLEAN

Interpolation + Normalisation

Integer wavenumber indices. Trapezium-rule area normalisation. Fingerprint region (630–880 cm⁻¹).

03 — REDUCE

Principal Component Analysis

251 wavenumber features → 20 principal components. sklearn PCA.

04 — TRAIN

k-NN Classifier

67:33 train/test split by compound (not spectrum). k=3 nearest neighbours.

05 — CLASSIFY

Unknown Identification

Personally collected spectra of unknown isomers classified against the full training library.

def load_spectra(n, spectra_dir="data/raw_ir_spectra"):
    spectra = []
    for entry in os.scandir(spectra_dir):
        df = pd.read_csv(entry.path, skiprows=4, sep=r"\s+",
                         names=[entry.name], index_col=0)
        df = normalisation(df)    # area → 1 via trapezium rule
        df = interpolation(df)    # decimal → integer wavenumbers
        df = narrow_range(df)     # keep 630–880 cm⁻¹ only

        name = re.sub(r'_\d+$', '', entry.name.removesuffix('.txt'))
        df.columns = [name]
        spectra.append(df)

    combined = pd.concat(spectra, axis=1)
    if n > 0:
        return perform_pca(n, combined)
    return combined
